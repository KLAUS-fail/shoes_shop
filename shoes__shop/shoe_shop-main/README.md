# 👟 Магазин обуви - Информационная система

Демонстрационный экзамен по компетенции "Веб-дизайн и разработка"

## 📋 О проекте

Информационная система для магазина обуви с разграничением прав доступа. 
Разработана в рамках демонстрационного экзамена.

### Функциональные возможности:

- **Авторизация** с ролями (Гость, Клиент, Менеджер, Администратор)
- **Просмотр каталога** товаров с подсветкой (скидки, наличие)
- **Фильтрация, поиск и сортировка** товаров (для менеджеров и админов)
- **Управление товарами** (CRUD) для администратора
- **Управление заказами** для менеджера и администратора
- **Автоматическое обновление остатков** при создании заказов

## 🚀 Технологии

- Python 3.12
- Django 6.0
- SQLite (разработка) / MySQL (продакшн)
- Bootstrap 5
- Pillow (работа с изображениями)
